# Hybrid Search QA Assistant

This project is a question-answering assistant that retrieves and generates responses to technical questions using a hybrid search approach (BM25 + Semantic Search) and RAG.

## Project Structure
- `notebook.ipynb`: Easy-to-run Colab notebook containing data loading, indexing, retrieval logic, evaluation, and RAG integration.
- `app.py`: Streamlit application for interactive, real-time Q&A.
- `backend.py`: Python module handling the BM25, MiniLM semantic search, and HuggingFace LLM initialization and inference.
- `evaluation_examples.md`: Examples of good and bad answers generated by the system.

## Team Log
1. **Milestone 1**: Built the BM25 lexical search using `rank_bm25` and tokenized text.
2. **Milestone 2**: Evaluated the BM25 search using MRR@10 and MAP@10.
3. **Milestone 3**: Integrated `sentence-transformers` (all-MiniLM-L6-v2) for semantic search. Fused BM25 and Semantic search results using Reciprocal Rank Fusion (RRF).
4. **Milestone 4**: Connected the retrieval system to a HuggingFace RAG pipeline (`google/flan-t5-large` or `TinyLlama` depending on VRAM capabilities) to synthesize responses with citations.
5. **Milestone 5**: Created evaluation examples checking citation quality and final output formatting.
6. **Milestone 6**: Built the Streamlit interface for an easy-to-use live demo.

## Instructions to Run

### Colab Notebook
1. Upload the `notebook.ipynb` file to Google Colab.
2. Upload the `top_ai_questions.json` and `top_datascience_questions.json` dataset files to the Colab environment.
3. Run all cells. It will install dependencies, build indexes, evaluate the models, and show the generated answers.

### Streamlit UI (Local or Server)
1. Install dependencies:
   ```bash
   pip install streamlit rank_bm25 sentence-transformers transformers torch numpy pandas
   ```
2. Update the `data_paths` in `app.py` to point to the correct absolute paths of your JSON datasets.
3. Run the app:
   ```bash
   streamlit run app.py
   ```
4. Access the UI in your browser (usually http://localhost:8501) and test out questions!
